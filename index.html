INDEPENDENT PROJECTS

Yelp Review Categorization - NLPGDG Hackathon, Buffalo, 2015

    Aggregated reviews for each business in the Yelp dataset (approx 1.6M reviews for 61K businesses) to provide 0 to 5 star ratings in four categories: food, service, value for money, and ambience. Each sentence in a review was determined to be in one of the four categories by using semantic similarity based on WordNet synsets. The semantic polarity of the same sentence was also calculated. Using this, the score for each business was calculated from all the reviews to give the category ratings.

Simulation of the motion of colliding particlesSummer 2014

    Developed an application to help simulate the motion (according to the Newton's laws) of colliding particles. Used event-driven simulation to avoid checking for collisions between all particles. The main backing data structure used was a priority queue which held future collision "events", ordered by time. Developed a visualization for the same using the libraries provided at introcs. This project was inspired by the discussions at Princeton's algorithms course and here.
